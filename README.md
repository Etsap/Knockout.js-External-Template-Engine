# What Is It

The Knockout.js External Template Engine extends the original Knockout.js jQuery template engine to allow you to load external templates.

# Prerequisities
* Knockout.js 1.2.0pre or later
* jQuery 1.5 or later
* jquery.tmpl.js (jQuery templating)
